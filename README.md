# Online Shop Backend

A scalable backend for an online shop built with **Node.js**, **Prisma**, and **PostgreSQL**.

## Features (Phase 1)
- User registration & authentication (planned)
- Product listing
- Shopping cart
- Prisma ORM with PostgreSQL

## Tech Stack
- Node.js
- Express
- PostgreSQL
- Prisma

## Setup
```bash
npm install
cp .env.example .env
npx prisma migrate dev
npm run dev
